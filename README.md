# Data-Profession-Survey

**The goal of the project is to uncover insights into:**
    Salary trends across data roles
    Popular programming languages
    Job satisfaction factors (salary, work-life balance, growth)
    Difficulty of breaking into the data industry
    Demographics and career backgrounds

**Using Excel and Power BI, the project involved**:
    Data cleaning and transformation using Power Query
    Data modelling for efficient analysis
    Building interactive dashboards for visual storytelling

**This project demonstrates my ability to**:
    Analyse real-world datasets
    Identify meaningful trends and patterns
    Communicate insights through data visualisation
    Support data-driven decision-making
